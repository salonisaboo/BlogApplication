# BlogApplication
This is a full-stack blog application built with Spring Boot for the backend, Angular for the frontend, and MySQL as the database.
## 🚀 Technologies Used
---
### 🔧 Backend: Spring Boot
- Spring Starter Web  
- Spring Data JPA  
- MySQL Driver  
- Lombok
### 🎨 Frontend: Angular
- Angular CLI  
- Angular Material  

### 🗄️ Database
- MySQL
